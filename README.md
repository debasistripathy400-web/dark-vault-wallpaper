# 🌑 DarkVault

**DarkVault** is a premium, high-fidelity wallpaper platform designed with a dark gothic aesthetic. It allows users to browse, download, and favorite mysterious and ethereal artworks across various categories including Anime, Nature, Technology, and 3D Art.

![DarkVault Preview](https://github.com/debasistripathy400-web/dark-vault-wallpaper/raw/main/preview.png)

## ✨ Features

- **Gothic Aesthetic:** A deep, mysterious UI with neon accents (Crimson & Cyan) and glassmorphism.
- **3D Hero Section:** Interactive 3D background elements using Three.js.
- **User Vault:** Secure login and registration with JWT authentication.
- **Favorites System:** "Lick" (Like) your favorite wallpapers to store them in your personal vault.
- **Dynamic Categories:** Seamless browsing through curated themes.
- **Profile Management:** Customize your bio and track your download/favorite history.
- **Responsive Design:** Optimized for both desktop and mobile "Shadow Hunters".

## 🚀 Tech Stack

### Frontend
- **React.js** (Vite)
- **Three.js / React Three Fiber** (3D Visuals)
- **Lucide React** (Iconography)
- **Axios** (API Communication)
- **CSS3** (Custom Glassmorphism & Animations)

### Backend
- **Django** (Python)
- **Django REST Framework** (API)
- **SimpleJWT** (Authentication)
- **SQLite** (Development Database)
- **Pillow** (Image Processing)

## 🛠️ Installation

### 1. Clone the Repository
```bash
git clone https://github.com/debasistripathy400-web/dark-vault-wallpaper.git
cd dark-vault-wallpaper
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/scripts/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
